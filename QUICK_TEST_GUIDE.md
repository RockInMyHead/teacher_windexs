# 🧪 Быстрый тест интерактивного AI учителя

## 🚀 Запуск

Проект уже запущен на: **https://teacher.windexs.ru/**

## ✅ Как протестировать

### Шаг 1: Откройте приложение

```
https://teacher.windexs.ru/
```

### Шаг 2: Выберите курс

Например:
- **Английский язык** (courseId: 1)
- **Русский язык** (courseId: 0)
- **Математика** (courseId: 2)

### Шаг 3: Нажмите "Начать обучение"

### Шаг 4: Ответьте на приветственные вопросы

**Пример для английского:**
```
Я учусь в 9 классе
Я изучал Past Simple
```

**Пример для русского:**
```
Я учусь в 10 классе
Я изучал Синтаксис сложного предложения
```

**Пример для математики:**
```
Я учусь в 8 классе
Я изучал квадратные уравнения
```

### Шаг 5: Получите интерактивный урок!

AI создаст структурированный урок с:
- 📚 Короткими блоками теории
- ✍️ Интерактивными тестами
- ✅ Итогами

## 🎯 Ожидаемый результат

После вашего ответа, AI должен вернуть JSON с уроком, который автоматически отобразится как интерактивная карточка с:

1. **Прогресс-баром** вверху
2. **Теоретическими блоками** (2-3 предложения)
3. **Тестами с 4 вариантами ответа**
4. **Мгновенной проверкой** правильности ответа
5. **Объяснениями** для каждого вопроса

## 🐛 Если что-то не работает

### AI не возвращает интерактивный урок?

Проверьте консоль браузера (F12). AI должен вернуть JSON массив.

### AI возвращает обычный текст?

Это нормально для первого приветственного сообщения. Интерактивный урок появляется после того, как вы ответите на вопросы о классе и последней изученной теме.

### Ошибка OpenAI API?

Убедитесь, что в `.env` файле правильно установлен `VITE_OPENAI_API_KEY`.
