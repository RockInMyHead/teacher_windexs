# Планы курсов (Course Plans)

## Описание

Файл `coursePlans.ts` содержит структурированные планы курсов для разных классов. Это основа для генерации персонализированных программ обучения на основе результатов тестирования.

## Структура данных

### CoursePlan
```typescript
interface CoursePlan {
  grade: number;        // Класс (1, 2, 3, ...)
  title: string;        // Название курса
  description: string;  // Описание курса
  lessons: LessonPlan[]; // Массив уроков
}
```

### LessonPlan
```typescript
interface LessonPlan {
  number: number;   // Номер урока
  title: string;    // Название урока
  topic: string;    // Тема урока
  aspects: string;  // Основные аспекты изучения
}
```

## API функции

### `getCoursePlan(grade: number): CoursePlan | null`
Возвращает план курса для указанного класса.

```typescript
const plan = getCoursePlan(1); // План для 1 класса
```

### `getLessonFromPlan(grade: number, lessonNumber: number): LessonPlan | null`
Возвращает конкретный урок из плана курса.

```typescript
const lesson = getLessonFromPlan(1, 5); // Урок 5 из 1 класса
```

### `getAvailableGrades(): number[]`
Возвращает массив всех доступных классов.

```typescript
const grades = getAvailableGrades(); // [1, 2, 3, ...]
```

## Использование

```typescript
import { getCoursePlan, getLessonFromPlan } from '@/utils/coursePlans';

// Получить план для 1 класса
const grade1Plan = getCoursePlan(1);

// Получить урок 10 из 1 класса
const lesson10 = getLessonFromPlan(1, 10);

// Показать информацию об уроке
if (lesson10) {
  console.log(`${lesson10.title}: ${lesson10.topic}`);
  console.log(`Аспекты: ${lesson10.aspects}`);
}
```

## Добавление новых планов

Для добавления плана нового класса:

1. Создайте объект `CoursePlan` с соответствующей структурой
2. Добавьте его в массив `COURSE_PLANS`
3. Обновите тесты в `coursePlans.test.ts`

```typescript
const newGrade2Plan: CoursePlan = {
  grade: 2,
  title: "Английский язык для 2 класса",
  description: "Продвинутый курс для второклассников",
  lessons: [
    // Массив уроков
  ]
};

COURSE_PLANS.push(newGrade2Plan);
```

## Тестирование

Файл включает полный набор unit-тестов в `coursePlans.test.ts`:

```bash
npm test coursePlans.test.ts
```

Тесты проверяют:
- Корректность структуры данных
- Функциональность API функций
- Валидность данных планов курсов
