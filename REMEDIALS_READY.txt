  └─ Создает промпт для OpenAI API
  └─ Содержит инструкции для AI

  insertRemedialsIntoLesson()
  └─ Вставляет ремедиалы в конец урока
  └─ Добавляет разделитель и финальный блок

🚀 ТАК ЭТО РАБОТАЕТ
════════════════════════════════════════════════════════════════════════════════

  1. Ученик начинает урок (15 основных блоков)
     └─ Инициализируется LessonProgressTracker
  
  2. Ученик выполняет практические блоки с вопросами
     └─ Каждый ответ записывается в tracker.recordAnswer()
     └─ Если ❌ → добавляется в список ошибок
  
  3. Ученик завершает все 15 блоков
     └─ Система анализирует ошибки
  
  4. Если есть ошибки:
     └─ Генерируется промпт для AI
     └─ OpenAI создает новые ремедиальные блоки
     └─ Ремедиалы вставляются в конец урока
  
  5. Урок превращается в:
     Блоки 1-15 (оригинальные)
     + Разделитель "🔄 Повторение трудного материала"
     + Ремедиальные блоки (1000+) с НОВЫМИ примерами
     + Финальный блок "✅ Отлично! Готовы продолжать"
  
  6. Ученик повторяет и закрепляет материал

  from '../utils/adaptiveLessonFlow';

// Инициализация
const tracker = new LessonProgressTracker('lesson-1', 15);

// Запись ответа

  2, 'practice', 'Модальные глаголы', 'modals', 'q1',
  'Choose must or should', 'must', 'should', 'beginner'
);

// Отображение статистики
<RemedialsSystemComponent tracker={tracker} />

// Генерирование ремедиалов
if (tracker.shouldAddRemedials()) {
  const prompt = generateRemedialsPrompt(tracker.getFailedBlocks(), 'English');
  const remedialsBlocks = await callOpenAI(prompt);
  const fullLesson = insertRemedialsIntoLesson(blocks, remedialsBlocks);
  setBlocks(fullLesson);
}
